# Instant Feedback Feature

The quiz now provides **instant feedback** after each answer!

## How It Works

### MCQ (Multiple Choice Questions)

- Click any option
- Immediately see if it's ✓ Correct or ✗ Wrong
- Correct option is highlighted in green
- Wrong selection is highlighted in red
- Cannot change answer once submitted

### Fill in the Blank

- Type your answer
- Click "Check Answer" button OR press Enter
- Immediately see if ✓ Correct or ✗ Wrong
- If wrong, the correct answer(s) are shown
- Cannot edit after submission

### Multiple Answers

- Select multiple options
- Last selection triggers automatic check
- All correct answers highlighted in green
- Wrong selections highlighted in red
- Cannot change selections after answered

## Score Tracking

- Score updates **in real-time** at the top
- Only correct answers count toward the score
- For multiple choice with penalty, partial credit is calculated
- Final score shown in the results modal
